# SNS運用代行 キーワード検索ボリューム調査

ラッコキーワードで「SNS運用代行」と検索して取得したキーワード一覧に、aramakijake.jp の月間検索ボリューム（Yahoo! JAPAN / Google）を追加したデータです。

## ファイル構成

| ファイル | 内容 |
|---|---|
| `sns運用代行_キーワード管理ページ.html` | 全キーワードを一覧管理できるHTMLページ（検索・フィルター・並び替え機能付き） |
| `sns運用代行_キーワード.csv` | 元データ（キーワード・重要度・カテゴリ） |

## データ概要

- **総キーワード数**: 245件
- **カテゴリ数**: 19カテゴリ
- **検索ボリュームデータソース**: [aramakijake.jp](https://aramakijake.jp/)
- **取得日**: 2026年4月29日

## HTMLページの機能

- キーワード検索（リアルタイム絞り込み）
- 重要度フィルター（＋ / ＋＋ / ＋α）
- カテゴリフィルター（19カテゴリ）
- 並び替え（Google・Yahoo 検索数 多い順/少ない順）
- Google検索ボリュームのバー表示
- キーワードクリックでGoogle検索に遷移
